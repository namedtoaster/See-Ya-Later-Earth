## Key Color Code (from drawings)

- **Red**: Paths / player routes / arrows
- **Orange**: Rooms / walls / boundaries
- **Blue**: Level boundaries / outer walls
- **Green**: Interactive elements
- **Purple dots**: Key locations / objectives
